# ptt-kurskarten

## Quickstart

```bash
npm install
```

```bash
npm run dev:api
```

```bash
npm run dev:web
```

Open the UI at http://localhost:4200 and the API at http://localhost:3000/api/v1.

## Localization (DE/FR)

Run a specific locale:

```bash
npm run dev:de
```

```bash
npm run dev:fr
```

Localized routes are served under `/de/` and `/fr/`.

Build a specific locale:

```bash
npm run build:de
```

```bash
npm run build:fr
```
